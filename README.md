## VisionSpatial (ML Kit & Orientación) 
Aplicación para Android que utiliza visión artificial para detectar objetos y personas en tiempo real, calculando su posición cardinal (Norte, Sur, Este, Oeste) relativa al usuario mediante sensores de orientación.

## Funcionalidades Clave
*   **Detección con ML Kit:** Identificación precisa de etiquetas como muebles, personas y objetos del entorno en tiempo real.
*   **Geolocalización Espacial:** Mapeo automático de la dirección cardinal del objeto detectado integrando la cámara con el sensor de brújula del dispositivo.
*   **Interfaz Dinámica:** Interfaz de usuario construida en XML para una visualización fluida de los resultados de detección.

## Stack Tecnológico
*   **Core:** Android SDK (Java/Kotlin).
*   **Inteligencia Artificial:** Google ML Kit (Object Detection API).
*   **Frontend:** XML Layouts & Android View System.
*   **Hardware:** Integración de CameraX y Sensores de Orientación.
